# Card Game Application

## Getting Started
```
# install global packages
$ npm install -g gulp react-native-cli mocha

# install gulp dependencies
$ npm install

# install server and client dependencies
$ gulp install
```

## Gulp Tasks
```
# Testing client/server
$ gulp test

# Testing client
$ gulp test-client

# Linting client/server
$ gulp lint

# Linting client
$ gulp lint-client
```
